# Waterdrop Codex Skills

水滴业务相关 Codex Skills 集合。每个一级目录对应一个可独立安装的 Skill。

## Skills

- `analyzing-sea-ai-badcases`：分析 sea.AI 机器人 Badcase，定位应用层、子 Agent 层和专家层配置，并按固定格式输出优化建议。

## 安装

在 Codex 中使用 Skill Installer，从本仓库安装指定子目录：

```text
仓库：Tsytt/Waterdrop
路径：analyzing-sea-ai-badcases
```

安装后使用：

```text
$analyzing-sea-ai-badcases
```

使用者需要自行具备相关飞书文档与 sea.AI 平台的访问权限及登录状态。
